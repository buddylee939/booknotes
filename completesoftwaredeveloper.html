Creating the Plan

Work backwards from the goal you want to achieve

  1. What you need to learn
  2. figure out the order of what you need to learn
  3. how you are going to learn it is important
  4. then, figuring out what you will need to do to prepare for applying for jobs
  5. getting your first job
  6. plan for getting the job
  7. where are you going to look?
  8. what are you going to do?
  9. what kinds of jobs are you going to apply for
  10. also a plan for how you are going to continue your personal development and education after getting your first job

How to figure out what options are there

  1. ask software developers you know
  2. what kind of things are you interested in creating and research some of the associated technologies and programming languages
  3. web apps? mobile apps? regulate fridge temperature? send astronauts to space?
  4. SEARCH INDEED FOR TECHNOLOGIES NEEDED OR REQUIRED

A CONCRETE EXAMPLE

  1. Someone who wants to become a web developer utilizing Node.js
  2. Goal: become a Node.js developer
  3. Plan:
  4. Learning:
  5. learn the basics of Javascript
  6. learn about web pages and web development technologies like HTML and CSS
  7. learn the basics of Node.js
  8. Be able to write some kind of simple Node.js web app
  9. learn about the different frameworks and technologies developers use to develop Node.js apps
  10. Fill in some frameworks or technologies to use with Node from research above
  11. learn some kind of database technology to use with Node
  12. learn computer science basics: algorithms/data structures
  13. learn best practices for writing good code
  14. learn ho wo design the architecture of a Node app

PREPARING FOR GETTING A JOB

  1. start looking at job descriptions for a Node developer in my area and find out what skills employers want
  2. come up with a list of companies, locally, that I can likely get a job at
  3. start attending user groups in the area
  4. start networking with other local Node devs
  5. hire a resume writer to help me write a good resume
  6. practice codeing interview questions
  7. practice mock interviews
  8. build a portfolio of a few apps to demo

GETTING A JOB

  1. contact all the people in my networks to let them know what value I can provide and what I am looking for
  2. start applying for junior-level jobs or internship programs
  3. plan to apply for at least 2 jobs each day
  4. debrief with myself after interview and decide what skills need to be worked on
